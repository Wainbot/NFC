## Installation

```bash
$ git clone https://github.com/Wainbot/NFC
$ cd NFC/
$ npm install
```

## Launch app on local

(in full page)

```bash
$ ionic serve
```

OR


(for android + ios)

```bash
$ ionic serve -l
```


## Build app on phone (android)

```bash
$ ionic run android
```
