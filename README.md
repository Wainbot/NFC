``

## Installation

``
$ cd NFC/
$ npm install
``

## Launch app on local

(in full page)
`` 
$ ionic serve 
``

OR

(for android + ios)
`` 
$ ionic serve -l
``

## Build app on phone (android)

`` 
$ ionic run android
``
